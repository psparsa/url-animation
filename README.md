# URL ANIMATION

![](https://s1.gifyu.com/images/url-animation.gif)


### Installation

Install the dependencies.

```sh
$ cd url-animation
$ npm install
```

run development server.

```sh
$ npm start
```
make build.

```sh
$ npm run build
```
