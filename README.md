# URL ANIMATION

![](https://s1.gifyu.com/images/url-animation.gif)


### Installation

Install the dependencies and devDependencies and start the server.

```sh
$ cd url-animation
$ npm install
```

run development server.

```sh
$ npm start
```
For make buid.

```sh
$ npm run build
```
